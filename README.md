# Destiny Pixel Dungeon

Destiny Pixel Dungeon is an open-source traditional roguelike dungeon crawler with randomized levels and enemies, and hundreds of items to collect and use. It's based on [Shattered Pixel Dungeon](https://shatteredpixel.com/shatteredpd/) by 00-Evan, which is in turn based on the [source code of Pixel Dungeon](https://github.com/00-Evan/pixel-dungeon-gradle), by [Watabou](https://watabou.itch.io/).

Destiny Pixel Dungeon is currently a work in progress so compiles for nothing.
